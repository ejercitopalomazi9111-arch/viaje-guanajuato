# Viaje a Guanajuato — Instituto Rembrandt de Querétaro

Mapa interactivo de la expedición STEAM al patrimonio histórico de Guanajuato e Hidalgo.
Proyecto institucional DA016 · Bachillerato Rembrandt · 2026.

## Stack

- HTML5 + CSS3 + Vanilla JavaScript
- [Leaflet 1.9.4](https://leafletjs.com/) — mapa interactivo
- [CartoDB Voyager](https://carto.com/basemaps/) — tiles OSM
- Google Fonts: Newsreader, Inter, JetBrains Mono
- Imágenes: Wikimedia Commons (CC-BY-SA)

## Contenido

- **12 paradas reales** del itinerario (4 Dolores Hidalgo + 8 Guanajuato Capital)
- **11 asignaturas** con sus rúbricas STEAM
- **6 materias con contenido pre-cargado** (Programación, Herramientas Digitales, Negocios BMC, Educación Física, Inglés, Humanidades, Literatura)
- **5 materias pendientes** de evidencias del viaje

## Desarrollo local

Abre `index.html` en cualquier navegador moderno. No necesita servidor.

## Despliegue

GitHub Pages desde la rama `main`.
